== Description ==

This is a Good Plug-in for Adult Sites... Contents Warning

Anyone who's Trying to access you're page (EXCEPTION in place to BOTS ) for the FIRST Time or After Cookie Expired or If Settings set, Will be Redirected to Page Specified in Settings. I have Also Included the Page That Redirecting (door.php).

This is a Very Good Tool for Showing a Warning, Agreement Message to Users.

== Installation ==

Copy the wp-door into wp-content/plugins/
Log-in as Admin
Go to : Plugins
	Activate WP-Door Plugin.

You are almost Done!

== Options ==

Go to: Settings - > Door Options

Here You can Set up Cookie Expiry (Never, Time or Always Ask (once per session)), Exception list of Bot names & Redirect URL.

== DOOR.PHP ==
Make Sure you Copy this / Rename to accommodate you. This has some Stand Alone PHP codings that Set Cookies & Session By Users Selection.
You can Edit this Contents to Change Message Look & Feel... You may also Redirect to One of you Existing page as long as it does the Cookie / Session settings right...
Have a Look at the Door.php For more Info on Setting Cookie and each... or Drop me a Mail on rajeevanuk@gmail.com
